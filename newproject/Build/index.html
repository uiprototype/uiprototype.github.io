1. Introdução

Neste protótipo pretende-se demonstrar algumas das principais funcionalidades e componentes que estão presentes na biblioteca do Bootstrap.

O Bootstrap é uma framework open-source utilizada mundialmente para design e desenvolvimento front-end de aplicações web e sites. Esta framework é actualmente a mais popular e utilizada para desenvolvimentos responsive e mobile first.

As principais vantagens de se adoptar por esta framework são a maior facilidade de desenvolvimento, estrutura responsive, consistência entre browsers e plataformas, liberdade de customização e a grande comunidade de suporte.

3.4. Tooltip

As Tooltips podem ser utilizadas em diversos componentes diferentes (sempre nesta forma de mouse over). Neste caso são apresentadas em botões porque normalmente é onde fazem mais sentido. Através do "data-placement" é facilmente definida a sua orientação.

data-toggle="tooltip" data-placement="left" data-placement="top" data-placement="bottom" data-placement="right"


<button type="button" class="btn btn-default" data-toggle="tooltip" data-placement="left" title="Tooltip on left">Tooltip on left</button>
<button type="button" class="btn btn-default" data-toggle="tooltip" data-placement="top" title="Tooltip on top">Tooltip on top</button>
<button type="button" class="btn btn-default" data-toggle="tooltip" data-placement="bottom" title="Tooltip on bottom">Tooltip on bottom</button>
<button type="button" class="btn btn-default" data-toggle="tooltip" data-placement="right" title="Tooltip on right">Tooltip on right</button>

<!-- js function is needed in each html page where we use tooltips -->
<script>
  $(function () {
    $('[data-toggle="tooltip"]').tooltip()
  })
</script>

7. Images

7.1. Responsive Images

A utilização de imagens no Bootstrap pode ser responsive caso seja utilizada a classe "img-responsive". Esta classe aplica o comprimento máximo, altura automática e display em bloco para que a imagem ocupe toda a área ou elemento em que está inserida.

img img-responsive


<img src="..." class="img-responsive"></img>

7.2. Image Shapes

Através das classes "img-rounded", "img-circle" e "img-thumbnail" é possível dar diversas formas a qualquer imagem.

img-rounded img-circle img-thumbnail


 <img src="..." class="img-rounded">
 <img src="..." class="img-circle">
 <img src="..." class="img-thumbnail">

7.3. Picture Element

O elemento <picture> informa o browser da versão da imagem que deve ser carregada de acordo com algum critério, como por exemplo o viewport size ou o tipo de imagem suportado.

picture source img


<picture>
 <source media="(min-width: 650px)" srcset="assets/img/kitten-large.png">
